### A few things I've built

**🩺 Multimodal Explainable AI for breast-cancer detection** — *Univ. of Fez*
Combined a ViT on mammograms with XGBoost on tabular BI-RADS data. Lifted F1 from 0.81 → 0.86 and ROC-AUC by +0.09 over single-modality baselines, then added a Grad-CAM + SHAP explainability layer validated against radiologist annotations.

**🔍 Causal-discovery reliability framework** — *CEA-Saclay*
Built a way to benchmark and rank causal-discovery methods (PCMCI+, constraint-based) on time series *without* a ground-truth graph — using super-uniformity of conditional-independence p-values and stability under subsampling instead of accuracy-vs-truth.

**📄 AI recruitment platform** — *BCG X*
Semantic CV parsing and candidate–role matching with a human-in-the-loop chatbot that turns recruiter feedback into training signal, so the model gets better with use. Result: +24.3% top-5 match precision, ~5.6 hours saved per role.

**👤 Facial recognition from scratch** — *prépa project*
Facial recognition built on raw linear algebra — covariance computation, eigendecomposition via SVD, and projection onto a reduced eigenspace, all hand-implemented in NumPy. The project that made me fall for the math under ML.
